# COURSE PRINCIPLES

## Mission

Build the world's best free, open-source Platform Engineering & AI Infrastructure course.

The course should be practical, modern, industry-relevant, and continuously improvable.

---

## Educational Philosophy

Learning happens by building.

Every topic should include:

* Understand
* Practice
* Apply
* Review
* Extend

Students should spend more time building than reading.

---

## Project-Based Learning

Every major module must produce something real.

Examples:

* Linux automation scripts
* Dockerized applications
* Kubernetes clusters
* Terraform infrastructure
* Monitoring dashboards
* AI inference platforms

A student should graduate with a portfolio, not just notes.

---

## Progressive Learning

Topics should move from:

Beginner → Intermediate → Advanced → Production

Never assume prior knowledge without teaching prerequisites.

---

## Real-World Focus

Every lesson should answer:

* Why does this exist?
* Where is it used?
* What problems does it solve?
* How do companies use it?

---

## Hands-on First

Theory should support practice.

Every lesson should contain practical exercises whenever possible.

---

## Production Quality

Examples should follow industry best practices.

Avoid toy examples when realistic alternatives exist.

---

## Open Source First

Prefer:

* Linux
* Docker
* Kubernetes
* Terraform
* Prometheus
* Grafana
* OpenTelemetry
* Ollama
* vLLM
* PostgreSQL
* Git

before introducing commercial alternatives.

---

## Free Learning

Recommend free resources whenever possible.

Official documentation should be the primary reference.

---

## Continuous Improvement

The course is a living project.

Lessons, labs, and projects should evolve with industry practices.

---

## AI-Assisted Learning

AI is used to:

* explain
* review
* generate exercises
* answer questions

AI should not replace critical thinking.

Students are encouraged to understand before using AI-generated solutions.

---

## Quality Over Quantity

One excellent lesson is better than ten mediocre lessons.

Every module should be reviewed before publication.
