# SRE Fundamentals Quiz

1. Which of the following defines a target value for a service level?
   - A) SLI
   - B) SLO
   - C) SLA
   - D) Error Budget
   **Answer:** B

2. What is the primary purpose of an error budget?
   - A) To track financial costs of downtime.
   - B) To punish developers for bad code.
   - C) To balance feature velocity with reliability.
   - D) To measure network latency.
   **Answer:** C

3. A blameless postmortem focuses on fixing:
   - A) People
   - B) Systems
   - C) Management
   - D) Clients
   **Answer:** B
