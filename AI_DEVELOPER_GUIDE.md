# Prompting Guide

Guidelines and best practices for crafting effective AI prompts for platform engineering workflows, agent interactions, and automation tasks.

## Key Principles
1. **Be Specific & Explicit:** State exact requirements, context, and desired output formats.
2. **Provide Context:** Give relevant architectural or system background.
3. **Structured Outputs:** Utilize JSON, YAML, or markdown tables for clean parsing.

## Prompt Templates
- Infrastructure as Code (IaC) Generation
- Automated Code & Architecture Review
- Runbook & Incident Analysis
